# IT Support Ticket Tracker

A recruiter-focused IT operations and support workflow platform built with Flask, Python, SQLite, and dashboard analytics.

## Overview

This application simulates the type of ticket management and operational workflow system used by internal technology support teams.

The platform allows users to:

- Create and manage support tickets
- Track ticket priority and status
- View operational dashboard metrics
- Review ticket history
- Monitor support workload
- Organize issue resolution workflows

The goal of this project is to demonstrate practical systems analysis, workflow management, and business technology concepts commonly used in IT operations environments.

## Technologies Used

- Python
- Flask
- SQLite
- HTML/CSS
- SQLAlchemy
